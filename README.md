El objetivo de este proyecto es mostrar que se saben usar bases de datos en MySQL, incluyendo:

Crear y organizar bases de datos relacionales

Crear tablas y relacionarlas usando claves primarias y foraneas

Cargar, leer, modificar y borrar datos (CRUD)

Hacer consultas usando JOIN entre varias tablas

Mantener los datos correctos usando relaciones y borrado en cascada
